# 資格対策ページ 問題集データ

NotebookLM等で作成した問題集をここに置くと、資格対策ページに自動反映されます。
ファイル名は `questions_{資格ID}.md`（または `.json`）。

## 資格IDと対応

| 資格ID | 資格名 |
|---|---|
| `g_kentei` | G検定 |
| `genai_passport` | 生成AIパスポート |
| `ds_kentei` | DS検定 |
| `ap` | 応用情報技術者試験（AP） |
| `python_ds` | Python 3 エンジニア認定データ分析試験 |

例：応用情報の問題集 → `questions_ap.md`

## Markdown形式（推奨）

`##` が問題、`###` が解答です。「問1」「Q1」等のラベルは付けても付けなくてもOK（表示側で採番します）。

```markdown
## 問1 ACID特性のうち原子性とは何か？
### 解答
トランザクションが全部成功か全部失敗のいずれかになる性質。

## 問2 RAID5の最低ディスク本数は？
### 解答
3台。パリティを分散配置し1台の故障に耐える。
```

## JSON形式（どちらでも可）

`.md` の代わりに `.json` を置いてもOK。`.json` があれば優先されます。

```json
[
  { "q": "ACID特性のうち原子性とは何か？", "a": "全部成功か全部失敗になる性質。" },
  { "q": "RAID5の最低ディスク本数は？", "a": "3台。" }
]
```

## 反映方法

ファイルを置いたら、アプリの資格対策ページで「🔄 更新」を押すか、ページを開き直すと反映されます。
ファイルが無い資格は、内蔵のサンプル問題（数問）が表示されます。
